# Памятка комиссии (устный экзамен)

Полный регламент — [README.md](README.md).

## Ход

1. Составить набор из **5** вопросов банка (И-*; У-* только как незадубливающая замена).
2. Студент отвечает **устно**, без устройств и материалов.
3. На каждый вопрос: при необходимости 1–2 уточнения → балл **0 / 0,5 / 1**.
4. Сумма по пяти вопросам: нужно **≥ 3,0**. В БРС: сумма × 4 (макс. 20).

## Быстрая шкала

| Балл | Смысл |
|---:|---|
| **1** | По существу; допущения; связное решение; риск/проверка; нет критической ошибки |
| **0,5** | Верное направление, но неполно / без связей / без критериев / заметные некритичные пробелы |
| **0** | Не по теме; грубая подмена понятий; противоречие кейсу; только названия продуктов; критическая ошибка без исправления |

Ориентир полноты — колонка «Ожидаемый ответ» в банке, не дословное совпадение.  
Стек: **рекомендуется**; обязательна инженерная логика.
