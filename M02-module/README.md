# Модуль 2. Источники и сбор данных

**Результат и КИУ:** РО-2, BD-2.2, базовый уровень.

**КИМ:** закрытые вопросы З-02, З-03 и З-17; один открытый вопрос из Т-05—Т-08 или Т-33—Т-36.

**Проверяется:** типы источников, API, выгрузки, прямое чтение БД и CDC, контракт источника, режим поступления, доступ и минимизация данных.

**Критерий:** студент выбирает способ получения данных по нагрузке, задержке, полноте, доступу и рискам; перечисляет обязательные элементы контракта источника.

Подробное содержание, контексты данных и ссылки на КИМ: [материалы модуля](materials.md).
