# Шкала перевода баллов в итоговую отметку

| Суммарный рейтинг за дисциплину | Итоговая отметка |
|---:|---|
| 0–59 баллов | неудовлетворительно |
| 60–74 балла | удовлетворительно |
| 75–89 баллов | хорошо |
| 90–100 баллов | отлично |

Положительная отметка выставляется при выполнении минимальных условий, указанных в [балльно-рейтинговой системе](points-rating-system.md): не менее 20 баллов за лабораторные работы, не менее **3,0 из 5** на устном экзамене (12 из 20 в БРС) и не менее 60 баллов суммарно.
