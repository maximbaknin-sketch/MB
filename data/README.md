# Исходные данные КРМ и ролей

## Файлы

| Файл | Назначение |
|---|---|
| [krm-v3.0.xlsx](krm-v3.0.xlsx) | первичная книга КРМ версии 3.0 (компетенции, индикаторы, уровни) |
| [roles.xlsx](roles.xlsx) | матрица «роли ↔ компетенции / уровни» (лист «Роли-компетенции наглядно») |

Источник: материалы команды дисциплины (файлы `КРМ-версия3.0.xlsx`, `Роли.xlsx`). В репозитории **не редактируются**.

## Листы `krm-v3.0.xlsx`

| Лист | Содержание |
|---|---|
| `Роли-компетенции наглядно` | уровни компетенций по ролям (в т.ч. Data Engineer) |
| `Роли - определения и функции` | определения ролей и трудовые действия |
| `Профессиональные` | компетенции, индикаторы, дескрипторы уровней |
| `Опережающие профессиональные` | перспективные компетенции |
| `Отраслевые` | отраслевое применение ИИ |
| `Универсальные (надпроф.)` | надпрофессиональные компетенции |
| `Аббревиатуры разные` | расшифровки кодов (BD = Big Data / работа с данными) |

## Индикаторы блока BD (для сверки ФОС)

По листу `Профессиональные` в книге есть:

| Компетенция | Индикаторы |
|---|---|
| BD-1 | BD-1.2, BD-1.3, BD-1.4, BD-1.5 (**BD-1.1 в книге отсутствует**) |
| BD-2 | BD-2.1, BD-2.2, BD-2.3 |
| BD-3 | BD-3.1, BD-3.2 |
| BD-4 | BD-4.1, BD-4.2, BD-4.3 |
| BD-5 | BD-5.1, BD-5.2, BD-5.3 |

Для роли **Data Engineer** на листе наглядной матрицы по блоку «Работа с данными»: BD-1 — С, BD-2 — С, BD-3 — С, BD-4 — П, BD-5 — П.

## Правило использования

Коды и формулировки переносить в РПД, карту РО–КИУ и КИМ **без изменения смысла** первоисточника. Предметный дескриптор курса добавлять отдельно.

Для модуля «источники и сбор» (РО-2) в КИМ используется индикатор **BD-2.2** (по книге КРМ).
