# Модуль 4. Концепции КХД

**Результат и КИУ:** РО-4, BD-3.2, средний уровень.

**КИМ:** закрытые вопросы З-05 и З-14; один открытый вопрос из Т-13—Т-16 или Т-41—Т-44.

**Проверяется:** DWH, Data Lake, Lakehouse, Data Fabric, Data Mesh, зоны данных и управление доступом.

**Критерий:** студент обосновывает архитектуру через потребителей, качество, ownership, историю и эксплуатационные ограничения.

Подробное содержание, контексты данных и ссылки на КИМ: [материалы модуля](materials.md).
