# Модуль 3. Модели хранилищ

**Результат и КИУ:** РО-3, BD-3.1, средний уровень.

**КИМ:** закрытый вопрос З-04; один открытый вопрос из Т-09—Т-12 или Т-37—Т-40.

**Проверяется:** Kimball, Inmon и Data Vault, гранулярность фактов, измерения, историзация и выбор модели хранения.

**Критерий:** студент сопоставляет подходы по стабильности источников, истории, аудиту и потребностям аналитиков, а не называет единственное универсальное решение.
