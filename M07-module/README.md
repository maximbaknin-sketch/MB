# Модуль 7. ETL/ELT-системы

**Результат и КИУ:** РО-6, BD-4.1, средний уровень.

**КИМ:** закрытый вопрос З-09; один открытый вопрос из Т-21—Т-24 или Т-49—Т-52; [ЛР-2](../Project/laboratory-works.md#лр-2-покритериальный-выбор-etlelt).

**Проверяется:** критерии выбора ETL/ELT, интеграционные подходы, матрица сравнения, источники ERP, CRM и системы прогноза.

**Критерий:** студент обосновывает выбор через функциональные, технические, ценовые и ИБ-критерии, а оценки альтернатив подтверждает проверяемыми основаниями.

Подробное содержание, контексты данных и ссылки на КИМ: [материалы модуля](materials.md).
