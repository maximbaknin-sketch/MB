# Модуль 9. Проектирование конвейеров данных

**Результат и КИУ:** РО-8, BD-4.2, продвинутый уровень.

**КИМ:** закрытые вопросы З-12, З-13 и З-20; дополнительный открытый вопрос из Т-61—Т-64; [ЛР-3](../Project/laboratory-works.md#лр-3-crm-интеграция-и-record-matching).

**Проверяется:** оркестрация, идемпотентность, наблюдаемость, BPMN, ручная и автоматическая выверка, прототип интеграции CRM.

**Критерий:** студент показывает устойчивый и проверяемый процесс: правила сопоставления, обработку ошибок, повторный запуск и наблюдаемость в пределах задания.
