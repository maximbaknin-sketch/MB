# Модуль 6. Хранилища для RAG

**Результат и КИУ:** РО-5, BD-5.2, продвинутый уровень.

**КИМ:** закрытый вопрос З-08; дополнительный открытый вопрос из Т-57—Т-60; [ЛР-1](../Project/laboratory-works.md#лр-1-гибридное-хранилище-для-rag-агента).

**Проверяется:** гибридное хранение документов и структурированных данных, RAG-контекст, C4, ER-диаграмма и обоснование стека.

**Критерий:** студент предъявляет и объясняет свой проектный артефакт; вопрос Т-57—Т-60 дополняет ЛР, но не заменяет её.

Подробное содержание, контексты данных и ссылки на КИМ: [материалы модуля](materials.md).
