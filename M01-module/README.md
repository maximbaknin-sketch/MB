# Модуль 1. Роль инженерии данных в ИИ

**Результат и КИУ:** РО-1, BD-2.1, базовый уровень.

**КИМ:** закрытые вопросы З-01 и З-18; один открытый вопрос из Т-01—Т-04 или Т-29—Т-32 из [реестра](../resources/problem-banks/open-and-exam-register.md).

**Проверяется:** назначение слоёв контура данных, роль Data Engineer, данные как актив и риск, перевод запроса потребителя в требования.

**Критерий:** студент корректно объясняет связи между источниками, интеграцией, качеством, хранением и ИИ-потребителем; различает зоны ответственности ролей и называет последствия ошибки.

Подробное содержание, контексты данных и ссылки на КИМ: [материалы модуля](materials.md).
