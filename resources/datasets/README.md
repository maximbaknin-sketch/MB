# Датасеты

Учебные наборы для лабораторных работ. В Git размещаются только синтетические / открытые материалы без персональных данных.

Формат карточки: [resource-card-template.md](../resource-card-template.md).

## ЛР-1. Недвижимость для гибридного RAG

| Поле | Содержание |
|---|---|
| Тип | датасет (учебный) |
| Название | Учебный контур «объявления и документы по недвижимости» |
| Аннотация | Студент формирует набор из открытых объявлений / смоделированных карточек объектов и учебных фрагментов документов (PDF/DOCX). При недоступности внешнего API допускается полностью синтетический набор с явным допущением в отчёте. |
| Связь с КИМ | [ЛР-1](../../Project/laboratories/LR-1-hybrid-rag-storage.md), модуль 6 |
| Доступ | собирается студентом; образец структуры полей — в методичке ЛР-1 |
| Лицензия | только открытые источники или собственная синтетика |
| Ограничения | без реальных ПДн клиентов; TIFF/сканы — только учебные |
| Дата проверки | 2026-07-23 |

## ЛР-3. Синтетические выгрузки двух CRM

| Поле | Содержание |
|---|---|
| Тип | датасет (синтетический) |
| Название | `crm_a_sample.csv` / `crm_b_sample.csv` |
| Аннотация | Обезличенные выгрузки с вариациями ФИО, адресов и заказов для отработки record matching (~15–16 строк каждая). |
| Связь с КИМ | [ЛР-3](../../Project/laboratories/LR-3-crm-integration.md), модуль 9; шаблон кода — [lr3_record_matching_template.py](../software/python-libs/lr3_record_matching_template.py) |
| Доступ | файлы в этой папке |
| Лицензия | CC0 (учебная синтетика команды курса) |
| Ограничения | не использовать как реальные клиентские данные |
| Дата проверки | 2026-07-23 |

## ЛР-4. Синтетическая телеметрия IoT

| Поле | Содержание |
|---|---|
| Тип | датасет (синтетический) |
| Название | `iot_telemetry_sample.csv` |
| Аннотация | Учебный поток показаний датчиков (температура, вибрация, давление) со статусами ok/warn/alarm/missing для проектирования зон Data Lake. |
| Связь с КИМ | [ЛР-4](../../Project/laboratories/LR-4-iot-data-lake.md), модуль 10 |
| Доступ | файл в этой папке |
| Лицензия | CC0 |
| Ограничения | без ПДн; не промышленный лог |
| Дата проверки | 2026-07-23 |
