# Карточка ресурса

Единый формат описания ресурса в каталоге `resources/`. Ниже — поля и заполненный пример.

## Поля

| Поле | Содержание |
|---|---|
| Тип | учебник / статья / датасет / бенчмарк / библиотека / сервис / промпт |
| Название | официальное название |
| Аннотация | 1–3 предложения о содержании и роли в курсе |
| Связь с КИМ | модуль, ЛР, код вопроса или вид контроля |
| Доступ | публичный URL, ЭБС, репозиторий или закрытый контур |
| Лицензия | условия использования |
| Ограничения | ПДн, атрибуция, запрет публикации исходников партнёра и т.п. |
| Дата проверки | ГГГГ-ММ-ДД |

## Пример

| Поле | Содержание |
|---|---|
| Тип | учебник |
| Название | Рис Дж., Хоусли М. Основы инженерии данных |
| Аннотация | Базовый учебник по жизненному циклу инженерии данных: источники, конвейеры, хранение, качество и эксплуатация. |
| Связь с КИМ | модули 1–10; теоретическая база закрытых и открытых вопросов |
| Доступ | фонд библиотеки / ЭБС вуза (уточняется ответственным за ресурсы) |
| Лицензия | издательская; цитирование по правилам ЭБС |
| Ограничения | полный текст в публичный репозиторий не копируется |
| Дата проверки | 2026-07-23 |
