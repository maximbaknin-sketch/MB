# Закрытые тесты

Для текущего контроля выбирайте **8–12** вопросов по завершённым модулям (см. календарь БРС). Ключ предназначен для преподавателя и исключается из версии для студентов.

Банк рассчитан на три точки тестирования без вынужденного пересечения блоков:

| Тест (БРС) | Модули | Рекомендуемый набор (пример) |
|---|---|---|
| Тест 1 | 1–3 | З-01…З-04, З-06, З-17, З-18, З-21…З-24 (**10**) |
| Тест 2 | 4–6 | З-05, З-07, З-08, З-19, З-25…З-32 (**12**) |
| Тест 3 | 7–10 | З-09…З-16, З-20, З-33…З-36 (**12**) |

Повтор тем между тестом и рубежным срезом допустим: тест проверяет факты, срез — связное обоснование.
