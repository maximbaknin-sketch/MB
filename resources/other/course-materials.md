# Материалы курса, использованные при разработке КИМ

| Группа материалов | Роль в репозитории |
|---|---|
| КРМ, ассесмент и ФОС | КИУ как единица измерения, связь заданий с индикаторами и уровнями, принципы банка вопросов |
| Методические материалы по лабораторным | постановки ЛР-1—ЛР-4, артефакты и вопросы к защите |
| Преподавание современного ИИ, LLM и агенты | контекст RAG, AI-агентов, CLI-инструментов и LiteLLM |
| Глубокое и машинное обучение, математика | предметная база для ИИ-потребителей данных |
| Инфраструктура для ИИ | программный стек, GPU, мониторинг, облачные PaaS/SaaS и эксплуатационные ограничения |
| Взаимодействие с индустрией и финансирование | предпроекты/PoC, гибкое ТЗ, ограничения данных и партнёрские кейсы |

Материалы не копируются в публичный репозиторий без проверки авторских прав и разрешения на распространение.
