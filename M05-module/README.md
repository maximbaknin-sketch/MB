# Модуль 5. Архитектуры данных для ИИ

**Результат и КИУ:** РО-5, BD-5.1, средний уровень.

**КИМ:** закрытые вопросы З-07 и З-19; один открытый вопрос из Т-17—Т-20 или Т-45—Т-48.

**Проверяется:** Feature Store, векторные БД, хранение признаков, метаданные, различие retrieval и генерации.

**Критерий:** студент выбирает класс хранилища по потребителю, версиям, доступу и воспроизводимости; не подменяет инженерное решение названием инструмента.

Подробное содержание, контексты данных и ссылки на КИМ: [материалы модуля](materials.md).
