# Модуль 8. Качество данных

**Результат и КИУ:** РО-7, BD-1.3, средний уровень.

**КИМ:** закрытые вопросы З-06, З-10 и З-11; один открытый вопрос из Т-25—Т-28 или Т-53—Т-56.

**Проверяется:** schema drift, свежесть, полнота, нормализация, record matching, валидация и решение о публикации данных.

**Критерий:** студент связывает метрику качества с потребителем, определяет правило контроля и описывает действие при нарушении.

Подробное содержание, контексты данных и ссылки на КИМ: [материалы модуля](materials.md).
